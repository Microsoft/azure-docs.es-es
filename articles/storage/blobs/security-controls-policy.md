# <a name="azure-policy-regulatory-compliance-controls-for-azure-data-lake-storage-gen2"></a>Controles de Cumplimiento normativo de Azure Policy para Azure Data Lake Storage Gen2

El [Cumplimiento normativo de Azure Policy](../../governance/policy/concepts/regulatory-compliance.md) proporciona definiciones de iniciativas creadas y administradas por Microsoft, conocidas como _integraciones_, para los **dominios de cumplimiento** y los **controles de seguridad** relativos a distintos estándares de cumplimiento. En esta página se enumeran los **dominios de cumplimiento** y los **controles de seguridad** para Azure Data Lake Storage Gen2.
Para que los recursos de Azure sean compatibles con el estándar específico, puede asignar las integraciones a un **control de seguridad** de manera individual.
