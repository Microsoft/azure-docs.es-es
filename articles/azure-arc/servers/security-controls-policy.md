# <a name="azure-policy-regulatory-compliance-controls-for-azure-arc-for-servers"></a>Controles de cumplimiento normativo de Azure Policy para Azure Arc para servidores

[Cumplimiento normativo de Azure Policy](../../governance/policy/concepts/regulatory-compliance.md) proporciona definiciones de iniciativas creadas y administradas por Microsoft, conocidas como _integraciones_, para los **dominios de cumplimiento** y los **controles de seguridad** relativos a distintos estándares de cumplimiento. En esta página se enumeran los **dominios de cumplimiento** y los **controles de seguridad** para servidores habilitados para Azure Arc (versión preliminar). Para que los recursos de Azure sean compatibles con el estándar específico, puede asignar las integraciones a un **control de seguridad** de manera individual.
