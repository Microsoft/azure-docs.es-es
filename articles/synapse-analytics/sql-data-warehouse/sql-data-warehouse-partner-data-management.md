# <a name="azure-synapse-analytics-data-management-partners"></a>Asociados de administración de datos de Azure Synapse Analytics

Para crear una solución de almacenamiento de datos con la característica del grupo de Synapse SQL, puede elegir entre una amplia variedad de herramientas líderes del sector. En este artículo se destacan las empresas asociadas de Microsoft con soluciones y herramientas de administración de datos compatibles con Azure Synapse.
